# CardNova

CardNova ist ein professioneller Frontend Prototyp für einen TCG Demo Store.

## Features

- Responsive Landingpage im Premium TCG Look
- Produktgrid mit Suche, Kategorie Filter und Raritätsfilter
- Produktdetails als Modal
- Lokale Demo Auswahl mit Mengenänderung und Summe
- Live Drop Bereich
- Newsletter Mockup
- Keine echte Bestellung und keine echte Zahlung

## Dateien

- `index.html` enthält die Seitenstruktur
- `style.css` enthält das komplette Design
- `script.js` enthält Filter, Produktdaten, Modals und Demo Auswahl

## GitHub Pages

Für eine statische Veröffentlichung kann GitHub Pages in den Repository Settings aktiviert werden:

Settings > Pages > Deploy from a branch > main > root

Danach ist die Seite über die GitHub Pages URL des Repositories erreichbar.
